## 1. Entrada, Processamento e Saída (EPS) ⚙️

É a base de qualquer programa:

- **Entrada (Input)**: Dados que o usuário fornece. <!-- .element: class="fragment" -->
- **Processamento**: O que o computador faz com os dados. <!-- .element: class="fragment" -->
- **Saída (Output)**: O resultado visualizado. <!-- .element: class="fragment" -->

---

## 2. Variáveis 📦

Espaços na memória para guardar dados.

- **Inteiro**: 1, 10, -5. <!-- .element: class="fragment" -->
- **Real**: 1.5, 3.14. <!-- .element: class="fragment" -->
- **Caractere**: "Olá", "A". <!-- .element: class="fragment" -->
- **Logico**: Verdadeiro/Falso. <!-- .element: class="fragment" -->

---

## 3. Operadores Aritméticos ➕

| Operação | Símbolo |
| :--- | :--- |
| Adição | `+` |
| Subtração | `-` |
| Multiplicação | `*` |
| Divisão | `/` |
| Resto (Mod) | `%` ou `MOD` |

---

## 4. Atribuição `<-` ⬅️

Guardando um valor na variável.

```visualg
nome <- "João"
idade <- 25
soma <- 10 + 20
```

---

## 5. Prática: Calcular Média 🧮

```visualg
Algoritmo "Media"
Var
   n1, n2, med : real
Inicio
   escreva("Digite a Nota 1: ")
   leia(n1)
   escreva("Digite a Nota 2: ")
   leia(n2)
   med <- (n1 + n2) / 2
   escreva("A média é: ", med)
Fimalgoritmo
```

---

## Resumo ✅

- Variáveis guardam dados. <!-- .element: class="fragment" -->
- EPS é o fluxo básico. <!-- .element: class="fragment" -->
- Use `<-` para atribuir valores. <!-- .element: class="fragment" -->
